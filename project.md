# Проект: Лендинг Жилина Елена

## Что это за проект

Персональный сайт-лендинг для Жилиной Елены — эксперта по оптимизации логистики и внедрению нейросотрудников в бизнес.
Сайт решает две задачи: запись клиентов на консультацию и продажа цифрового продукта.
Заявки с формы приходят напрямую в Telegram-бот (@ZhilinaLog_bot).

## Для кого

- Владельцы бизнеса и руководители логистических отделов
- Предприниматели малого и среднего бизнеса
- Рынок: Россия и СНГ

## Цифровой продукт (планируется)

**«Нейросотрудник за 7 дней»** — пошаговый мини-курс по внедрению ИИ-ассистента в логистику или малый бизнес без технических знаний.
Формат: PDF + видео. Целевая цена: 3 000–7 000 ₽.

## Telegram-бот (@ZhilinaLog_bot)

- Сейчас: принимает заявки с формы на сайте
- Планируется: авто-ответы клиентам, напоминания, воронка продаж

## Стиль дизайна

- **Тема:** тёмная (dark mode)
- **Цвета:** фон `#0d0f14`, акцент фиолетовый `#6c63ff`, бирюзовый `#00d4aa`
- **Типографика:** Segoe UI / system-ui, жирные заголовки, clamp() для адаптива
- **Элементы:** градиентные кнопки, карточки с hover-эффектом, glassmorphism в навигации
- **Анимации:** едущие машины (фуры, контейнеровозы, газели) на главном экране
- **Адаптив:** мобильные (от 360px), планшеты, десктоп
